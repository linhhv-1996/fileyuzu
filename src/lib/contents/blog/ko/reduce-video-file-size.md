## 왜 동영상 파일은 이렇게 클까요?

현대 카메라와 스마트폰은 최고의 화질로 영상을 저장하도록 설계되어 있습니다. iPhone이나 갤럭시로 찍은 영상은 기본적으로 높은 비트레이트(bitrate)로 인코딩되는데, 비트레이트란 1초당 저장되는 데이터의 양을 의미해요.

비트레이트가 높을수록 화질은 좋지만 파일 크기도 커집니다. 예를 들어:

- 1080p 30fps 영상: 약 60~130MB/분
- 4K 30fps 영상: 약 350~400MB/분
- 4K 60fps 영상: 약 600~800MB/분

이 수치들은 원본 파일 기준입니다. 공유하거나 웹에 업로드하기 위해서는 훨씬 작은 크기로 줄여야 합니다.

## 기존 방법의 문제점

### 온라인 압축 사이트의 함정

인터넷에는 수많은 "무료 온라인 동영상 압축" 서비스가 있습니다. 하지만 대부분의 서비스는 동일한 방식으로 작동해요:

1. 내 동영상 파일을 **해외 서버에 업로드**
2. 서버에서 일괄 처리 후 결과 파일 다운로드
3. 서버에 내 영상이 남아있을 가능성 존재

개인적인 영상, 업무용 자료, 혹은 저작권이 있는 콘텐츠를 낯선 서버에 올리는 것은 보안 측면에서 좋지 않습니다. 게다가 인터넷 속도에 따라 업로드에만 수 분이 걸리기도 하죠.

### 프로그램 설치의 번거로움

HandBrake나 FFmpeg 같은 전문 프로그램을 사용하면 세밀한 설정이 가능하지만, 일반 사용자가 코덱, 비트레이트, CRF 값 등을 직접 조정하는 것은 쉽지 않습니다. 잘못된 설정으로 오히려 화질만 망가뜨리는 경우도 많아요.

### 브라우저에서 직접 처리, 업로드 없음

가장 중요한 차이점입니다. UploadLess는 **파일을 서버에 보내지 않습니다.** 모든 압축 처리가 사용자의 기기, 즉 브라우저 안에서 이루어집니다. 파일이 기기 밖으로 나가지 않으니 개인정보 걱정이 없어요.

### 두 가지 압축 모드

**자동 최적화 모드 (기본값)**

가장 빠르고 간편한 방법이에요. 파일을 드래그해서 올리면 도구가 자동으로 영상을 분석해 화질과 용량의 균형이 가장 좋은 설정을 찾아 압축합니다. 뭘 설정해야 할지 모르겠다면 이 모드를 사용하면 됩니다.

**목표 파일 크기 모드**

"50MB 이하로 줄여야 해"처럼 구체적인 용량 목표가 있을 때 사용하는 모드입니다. 자동 최적화를 끄고 원하는 목표 크기(MB)를 입력하면, 그 크기에 맞춰 압축해 드립니다. 이메일 첨부 용량 제한이나 플랫폼 업로드 제한에 딱 맞게 맞출 때 특히 유용해요.

3. 자동 최적화 또는 목표 파일 크기 모드 선택

## 어떤 상황에서 동영상 용량 줄이기가 필요할까요?

### 카카오톡 / 메시지 전송

카카오톡의 일반 파일 전송 용량 제한은 300MB입니다. 하지만 모바일 환경에서는 그보다 훨씬 작은 파일도 전송이 느리거나 실패하는 경우가 있어요. 10~30MB 이하로 줄이면 빠르게 전송할 수 있습니다.

### 이메일 첨부

대부분의 이메일 서비스는 첨부 파일 용량이 25MB 이하로 제한되어 있습니다. 긴 영상도 목표 파일 크기 모드로 20MB 이하로 압축하면 이메일로 바로 보낼 수 있어요.

### 인스타그램 / 유튜브 업로드

SNS 플랫폼마다 권장 파일 크기와 해상도가 다릅니다. 업로드 전에 미리 최적화해두면 업로드 속도도 빠르고 플랫폼의 자체 재인코딩으로 인한 화질 저하도 최소화할 수 있어요.

### 저장 공간 확보

오래된 영상을 보관용으로 압축해두면 스마트폰이나 컴퓨터의 저장 공간을 효율적으로 활용할 수 있습니다.

## 자주 묻는 질문

**Q. 압축하면 화질이 많이 나빠지나요?**

자동 최적화 모드는 화질 저하를 최소화하는 설정으로 압축합니다. 원본의 80~90% 수준의 화질을 유지하면서 파일 크기를 50~70%까지 줄이는 것이 일반적입니다. 물론 극단적으로 작은 크기로 압축할수록 화질은 더 많이 떨어집니다.

**Q. 어떤 파일 형식을 지원하나요?**

MP4, MOV, AVI, MKV 등 일반적으로 사용되는 주요 동영상 형식을 지원합니다.

**Q. 파일 크기 제한이 있나요?**

브라우저에서 직접 처리하는 방식이라 기기의 메모리와 성능에 따라 처리 가능한 파일 크기가 달라질 수 있습니다. 일반적인 사용 환경에서는 수 GB 크기의 파일도 처리 가능합니다.

**Q. 압축된 파일은 어디에 저장되나요?**

압축이 완료되면 기기의 기본 다운로드 폴더에 저장됩니다. 파일은 UploadLess 서버에 저장되지 않습니다.

**Q. 완전 무료인가요?**

네, 무료로 사용할 수 있습니다.
